Centos 7 安装 RocketMQ

1. MQ下载目录

选择适合的版本

2. 解压

unzip rocketmq-all-5.1.2-bin-release.zip

3. 修改内存, 自带的太大

修改为1G

vim /bin/runserver.sh
vim /bin/runbroker.sh

云服务器还需要修改broker.conf

新增 brokerIP1=公网IP

4. 启动

sh mqnamesrv
sh mqbroker -n 119.91.252.25:9876 -c /root/mq/rocketmq/conf/broker.conf
